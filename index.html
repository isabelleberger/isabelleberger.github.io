About

I am a postbaccalaureate fellow in the Oliver Lab, NIDDK, National Institutes of Health. Our lab is working to build predictive models of gene regulatory networks in Drosophila melanogaster. For this project I have assembled an S2 cell context-specific network model to be used as a prior, using publically available data from FlyBase, modENCODE, and the Sequence Read Archive.

My background is math, but since coming to the Oliver Lab I have been doing bioinformatics work and developing skills in data munging, visualization and analysis, primarily using Python. Other skills/areas of interest include machine learning, mathematical modeling, and some limited experience with R, Matlab, and Maple. You can find most of my code and more about my projects with the Oliver Lab on my Github page (https://github.com/isabelleberger).
