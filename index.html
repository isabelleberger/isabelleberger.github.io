About

I am a postbaccalaureate fellow in the Oliver Lab, NIDDK, National Institutes of Health. Our lab is working to build predictive models of gene regulatory networks in Drosophila melanogaster. For this project I have assembled an S2 cell context-specific network model to be used as a prior, using publically available data from FlyBase, modENCODE, and the Sequence Read Archive.

My background is math, but since coming to the Oliver Lab I have been doing bioinformatics work and developing skills in data munging, visualization and analysis, primarily using Python. Other skills/areas of interest include machine learning, mathematical modeling, and some limited experience with R, Matlab, and Maple. You can find most of my code and more about my projects with the Oliver Lab on my Github page (https://github.com/isabelleberger).

Data

Here you can find datasets I've assembled available for download.

Motifs

Bed file containing putative motifs for 115 transcription factors in Drosophila melanogaster. Motifs were mapped via FIMO using position weight matrices from the five fly motif databases on the MEME Suite webpage. Results were filtered to only include motifs located in the introns or the region 1kb upstream of the gene. The bed file contains location, the FBgn of the transcription factor, and the pvalue from FIMO.

  • TF motifs

S2 cell ChIP-Seq Peaks

Bed file containing peak calls for 21 transcription factors in Drosophila melanogaster. All available S2 cell transcription factor ChIP datasets on the Sequence Read Archive were downloaded and post-processed using a standardized workflow. Peaks were called using both macs2 and spp, reflect the current FlyBase release (dm6) and have been filtered for phantom peaks (as described in Jain et al 2014). As with motifs, only peaks located in the introns or 1kb upstream were included. The bed file contains location, the FBgn of the transcription factor, and the -log 10 q value from the peak caller.

  • TF peaks

    • Bed file containing peak calls for 8 histones in Drosophila melanogaster. Peaks were obtained by the same methods as transcription factors, described above. No location filtering has been done on this dataset. The bed file contains location, the name of the histone, and the -log 10 q value from the peak caller.

  • Histone peaks
